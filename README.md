# 🎥 Quran Reels Generator | صانع الريلز القرآني

![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![FFmpeg](https://img.shields.io/badge/FFmpeg-Required-green)
![License](https://img.shields.io/badge/License-Open_Source-orange)

**أداة احترافية مفتوحة المصدر لإنشاء مقاطع فيديو قرآنية قصيرة (Reels/Shorts/TikTok) تلقائياً بجودة عالية، مصممة لتعمل بكفاءة قصوى على الهواتف (Termux) والكمبيوتر.**

---

### 🚀 مميزات الإصدار (V2.0)
* **🛡️ فلتر إسلامي آمن (Safe Search):** نظام بحث ذكي وحصري يستبعد "النساء والأشخاص" تماماً من الخلفيات، ويركز البحث على الطبيعة والمساجد فقط لضمان شرعية المحتوى.
* **⚡ سرعة تصدير خيالية (Ultrafast):** تم ضبط إعدادات المعالجة لتصدير الفيديو في ثوانٍ معدودة دون استهلاك موارد الجهاز.
* **🎨 خلفيات Pexels المتجددة:** ربط مباشر مع مكتبة Pexels لجلب فيديوهات خلفية عالية الجودة وغير مكررة في كل مرة.
* **🎲 الاختيار العشوائي الذكي:** زر سحري يختار لك (القارئ + السورة + الآيات) بناءً على خوارزمية تضمن أن تكون مدة الفيديو مناسبة للريلز (أقل من دقيقة).
* **🔊 هندسة صوتية سلسة:** استخدام تقنية (Crossfade) لدمج تلاوة الآيات بذكاء، مما يمنع "التقطيع" أو السكتات المزعجة بين الآيات المتتالية.
* **💾 نظام التخزين المؤقت:** لا يملأ مساحة هاتفك؛ يقوم بحذف الفيديوهات القديمة تلقائياً ويحفظ الفيديو الحالي مؤقتاً للتحميل فقط.
* **📱 واجهة متجاوبة:** تصميم عصري يعمل بسلاسة على الموبايل، ويدعم الكتابة اليدوية لأرقام الآيات.

---

### ⚙️ كيف تعمل الأداة؟
تعمل الأداة من خلال دورة حياة برمجية منظمة:
1.  **الواجهة (UI):** صفحة ويب خفيفة للتحكم الكامل.
2.  **الخادم (Backend):** مبني بـ **Flask** لاستقبال الطلبات.
3.  **جلب البيانات:**
    * النصوص: `api.alquran.cloud` (الرسم العثماني).
    * الصوت: `everyayah.com` (MP3 بجودة عالية).
    * الخلفيات: `Pexels API` (فيديو عمودي Portrait).
4.  **المعالجة (Processing):** دمج الصوت والصورة والنص باستخدام **MoviePy** و **FFmpeg**.

---

### 🛠️ المتطلبات التشغيلية
قبل البدء، تأكد من أن مجلد مشروعك يحتوي على:
1.  `main.py`: (الكود الأساسي).
2.  `UI.html`: (ملف الواجهة).
3.  `requirements.txt`: (المكتبات).
4.  📁 مجلد **`fonts`**: (يجب أن يحتوي على ملف خط عربي `Arabic.ttf` وملف إنجليزي `English.otf`).

### 📥 دليل التثبيت والتشغيل (Termux / Linux)

**1. تحديث الحزم وتثبيت الأساسيات:**
```bash
pkg update && pkg upgrade -y
pkg install python ffmpeg imagemagick git -y


**2. تحميل المشروع (انسخ الرابط الخاص بك):**

```bash
git clone https://github.com/YOUR_USERNAME/Quran-Reels-Generator.git
cd Quran-Reels-Generator
```

**3. تثبيت مكتبات بايثون:**

```bash
pip install -r requirements.txt
```

**4. تشغيل الأداة:**

```bash
python main.py
```

**5. الاستخدام:**

* سيظهر لك رابط محلي (مثل `http://127.0.0.1:5000`). افتحه في متصفح كروم.
* أدخل مفتاح **Pexels API** (مجاني).
* ابدأ في صناعة المحتوى!
